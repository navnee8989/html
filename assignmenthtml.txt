9.How is Cell Padding different from Cell Spacing?

-->Cellpadding:

Cellpadding specifies the space between the border of a table cell and its contents (i.e) it defines the whitespace between the cell edge and the content of the cell.

Syntax:

<table cellpadding="value" >.....</table>
where, value determines the padding 
(space between the border of a table and its content)

-->Cellspacing 
specifies the space between cells (i.e) it defines the whitespace between the edges of the adjacent cells.

Syntax:

<table cellspacing="value" >.....</table>
where, value determines the padding

14.What is uss of span tag?Explain with Example

-->The <span> tag is an inline container used to mark up a part of a text, or a part of a document. The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute. The <span> tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.

Example:-
<span>
		<h1>hello world</h1>
</span>
